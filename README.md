# 图书管理系统 python440

本项目是一个基于 Python Django 与 MySQL 的图书管理系统，旨在为图书馆提供便捷的信息化管理服务。

## 技术栈

- 后端：Python Django
- 数据库：MySQL

## 功能模块

- **图书信息浏览**：用户可浏览图书的基本信息。
- **图书借阅**：用户可进行图书借阅操作。
- **图书归还**：用户可在系统中归还图书。
- **个人信息管理**：用户可管理自己的个人信息。
- **最佳读者**：系统根据借阅行为评选最佳读者。
- **借阅排行**：显示图书借阅的排行情况。
- **问题反馈**：提供用户反馈问题的渠道。

### 管理员功能

- **读者管理**：管理用户信息。
- **图书管理**：管理图书信息。
- **借阅管理**：管理借阅信息。
- **归还管理**：管理图书归还情况。
- **问题处理**：处理用户反馈的问题。

## 系统角色

- 普通用户：可进行图书浏览、借阅、归还和个人信息管理。
- 管理员：拥有系统全面管理权限。

## 运行环境

- Python 环境配置
- MySQL 数据库

## 部署步骤

1. 配置 Python 运行环境。
2. 安装 MySQL 数据库，导入数据。
3. 安装 Django，配置项目。
4. 部署项目到服务器。
